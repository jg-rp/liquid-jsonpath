## Installation

Install Python Liquid Babel using [pip](https://pip.pypa.io/en/stable/getting-started/):

```console
python -m pip install -U liquid-jsonpath
```

Or [pipenv](https://pipenv.pypa.io/en/latest/):

```console
pipenv install liquid-jsonpath
```
